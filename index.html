Predicting Steam Game Sales

The Problem domain was to identify if a game was going to go on sale on the steam game platform based on the available dataset. These dataset includes the developer info, the publisher info, total nunber of owners, average number of days between each sale, etc..Logistic regression, SVM and bagging classifier algorithms was used to categories data into several discrete classes. The dataset was furthur divided into testing, training and validation dtaset. The bagging classifer produced the highest oprediction accuracy of approximately 85%. =
